# LvcidPsyche/auto-browser  ·  ⭐655  ·  watch
https://github.com/LvcidPsyche/auto-browser · pushed 2026-06-29 · triaged 2026-06-30

**What it is:** An open-source, MCP-native, self-hosted browser agent with a human-in-the-loop (Playwright + noVNC + FastAPI + Docker, MIT). Gives an AI agent a real browser it can drive, with a human able to watch/intervene.
**Reusable for us:** A capability building block (browser for agents via MCP), not a token/effectiveness tool. Useful as a reference `mcp/` option when a project needs real browser automation locally.
**Token / effectiveness angle:** Neutral/indirect — the human-in-the-loop + visible session can save wasted agent loops on flaky web tasks.
**How to adopt:** Watch — lower stars, young. Revisit if a project needs self-hosted browsing; compare against Cloudflare Browser Rendering (hosted) before standardizing.
